# Nexus Mobile TTS — iOS

On-device text-to-speech companion app for iPhone and iPad.

## Status

**Coming Soon** — iOS development is in progress. The app will be distributed via TestFlight for beta testing before App Store release.

## Planned Features

- Kokoro TTS via CoreML (optimized for Apple Neural Engine)
- Background audio playback with lock screen controls
- AirPlay streaming to external speakers
- Handoff support between iPhone, iPad, and Mac
- Siri Shortcuts integration

## Requirements

- iOS 17 or newer
- iPhone 12 or newer (A14 Bionic or later)
- ~200MB storage for app + model
