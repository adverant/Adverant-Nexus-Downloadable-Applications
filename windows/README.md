# Nexus Local Compute — Windows

Run local AI models on Windows with NVIDIA GPU acceleration.

## Status

**Coming Soon** — Windows development is in progress.

## Planned Features

- CUDA-accelerated TTS inference (NVIDIA RTX 2060+)
- System tray application with quick controls
- OpenAI-compatible API on localhost:8880
- Automatic model downloading and management
- Support for Kokoro, Qwen3-TTS, and other models

## Requirements

- Windows 10 or 11
- NVIDIA GPU with CUDA 12+ support (RTX 2060 or newer)
- 8GB VRAM recommended
- ~2GB storage for app + models
